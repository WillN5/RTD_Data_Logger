# RTD_Data_Logger
10 Channel PT100 Data Logger<br />

4 wire PT100 measurement <br />
10 Hz sample rate per channel<br />
10 channels sequential read<br />
Log to SD card<br />
Display real-time to screen<br />
USB powered<br />
Battery power (in testing) <br />
